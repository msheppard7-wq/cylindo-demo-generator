Cylindo Demo Generator

Generate branded product demos in seconds

Auto-pull product data from Cylindo Content API, generate a branded PDP with 3D viewer and tear sheet, deploy instantly.

How it works
1

Gather inputs

You need 4 things: customer-id from CMS Settings, product code(s) from the product page, curator code from the embed snippet, and the brand name.

2

Run the command

Use the CLI tool or the /cylindo-demo Slack command. Product features and options are auto-fetched from the Content API.

3

Share the demo

Get a live URL with the branded PDP, interactive 3D viewer, fabric/material selectors, and a downloadable tear sheet — all auto-generated.

Slack command
/cylindo-demo 4404 "HAVEN COMFORT ARM SOFA,NORFOLK WIDE ARM PLEATED SOFA" a7ap4vak "Serena & Lily"
CLI usage
node generate.js \
  --customer 4404 \
  --products "HAVEN COMFORT ARM SOFA,NORFOLK WIDE ARM PLEATED SOFA" \
  --curator a7ap4vak \
  --brand "Serena & Lily"
